Jimeng Free API Windows 便携版
=============================

一、系统要求

- Windows 10 或 Windows 11，64 位 x64 系统。
- 无需安装 Node.js、npm 或 Docker。
- 请先完整解压 ZIP，再启动程序；不要直接在压缩包预览窗口中运行。

二、启动与停止

1. 双击“启动服务.bat”。
2. 启动成功后浏览器会打开 http://localhost:8001 。
3. 首次访问请按页面提示设置管理员账号和密码。
4. 需要退出时双击“停止服务.bat”，不要直接删除正在运行的目录。
5. “查看日志.bat”可打开启动输出和错误日志。

重复双击启动脚本不会创建第二个服务进程。

三、端口和浏览器配置

用记事本打开 config\portable.env：

PORT=8001
HOST=0.0.0.0
AUTO_OPEN_BROWSER=1

- PORT：本机监听端口，范围 1 到 65535。
- HOST：监听地址。仅本机使用可改为 127.0.0.1。
- AUTO_OPEN_BROWSER：1 表示启动后打开浏览器，0 表示不打开。

修改后请先停止服务，再重新启动。若提示端口被占用，请更换 PORT；
启动器不会结束占用端口的其他程序。

四、数据与更新

- data\：管理员密码、调用统计和媒体记录，必须备份。
- logs\：运行日志。
- config\portable.env：用户配置。

更新便携包时，先停止服务，再备份并复制上述 data、logs 和配置文件。
整个目录可以移动到其他位置，支持带空格和中文的路径。

五、即梦 Session ID

本包不内置任何账号凭据。生成图片或视频时，需要使用自己的即梦
sessionid，并在兼容 OpenAI 的客户端中设置：

Authorization: Bearer sessionid_value

多个账号可用英文逗号分隔。请妥善保管 Session ID，不要上传或分享。

六、故障排查

- 启动失败：查看 logs\service.stderr.log。
- 页面打不开：确认启动窗口显示成功，并检查 PORT。
- 提示文件缺失：重新完整解压 ZIP，避免被安全软件隔离文件。
- 无法停止：不要手动修改 run\service.pid；可先查看任务管理器。
